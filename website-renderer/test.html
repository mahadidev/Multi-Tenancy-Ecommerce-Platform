How it works

  1. Dashboard Sidebar: When user clicks "Visit Website" button
  2. URL Generation: Button generates http://localhost:3001/?subdomain=store-name
  3. Website Renderer: Next.js app receives subdomain parameter
  4. API Call: Renderer fetches data from Laravel: GET /api/website/render/{subdomain}
  5. Rendering: Renderer displays the complete website with all components

Troubleshooting

If you see React Router errors like No routes matched location "/websites/1":

  • The buttons now use window.open() instead of anchor tags
  • This prevents React Router from intercepting the navigation
  • Check browser console for debugging information
